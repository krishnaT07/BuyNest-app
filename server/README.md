# BuyNest Server

Backend API for the BuyNest hyperlocal e-commerce platform.

## Setup

This backend will be implemented using Supabase for:
- Database management
- Authentication
- Real-time subscriptions
- File storage
- Edge functions

## Planned Features

- User authentication and authorization
- Product management APIs
- Order processing
- Seller onboarding
- Admin dashboard APIs
- Location-based search
- Real-time notifications
